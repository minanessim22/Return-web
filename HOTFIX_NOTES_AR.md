# HOTFIX v4

التعديلات في هذه النسخة:

- رفع عتبة الـ matching التلقائي إلى **80%**
- أي بلاغ يدخل حالة **MATCHED** يظهر تحت تبويب **Found Matches**
- تم استبعاد الحالات `MATCHED / RESOLVED / CLOSED` من **My Reports** حتى تظل قائمة التقارير النشطة نظيفة
- تحسين الاحتفاظ بالجلسة:
  - مدة أطول
  - تجديد تلقائي عبر `/api/me`
  - keep-alive دوري في الواجهة
  - عدم تسجيل الخروج عند فشل request عابر
- تحسين صفحة **Chat**:
  - polling دوري للتحديث
  - optimistic sending
  - refresh يدوي
  - تقليل رسائل الفشل العارضة
- تحويل **Devices** من واجهة شكلية إلى إدارة فعلية:
  - عرض الأجهزة الحقيقية من الباك إند
  - إنشاء جهاز جديد QR / NFC / GPS
  - ربط الجهاز بملف تعريف
  - تعديل الحالة، البطارية، الموقع، وفترة التحديث
  - تحديثات عبر API حقيقية
- مسار `/devices` والتاب العلوي للأجهزة يفتحان الآن **لوحة الأجهزة الفعلية** بدل الشاشة الوهمية

## تشغيل المشروع

```bash
npm install
npm run dev
```

## ملاحظات

- هذه النسخة مبنية على **نفس المشروع السابق** مع hotfix إضافي، وهي النسخة الأحدث التي يجب استخدامها.
- الإيميل الحقيقي ما زال يحتاج إعداد مزود بريد في `.env.local`.
